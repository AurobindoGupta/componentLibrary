How to Use

You can either import the entire Component Library or individual components by using the Links given bellow in your code as shown bellow

Component Library Link- https://panda-ui-neog.netlify.app/Components/components.css

Individual Component Link- https://panda-ui-neog.netlify.app/components/alert/styles.css

As seen in the above example with the alert component, if you want to import some other component just change the name of the component -"alert" in the link above.
